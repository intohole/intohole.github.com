安装
-------------
+ 打开[Haroopad](http://pad.haroopress.com/user.html)页面
+ 找到自己系统版本对应版本；
		
        #命令行下敲击命令，查看自己系统版本
    	$ uname -a 
+ 下载对应版本的deb版本
		
        #命令行下安装命令
        $ sudo dpkg -i haroopad-v0.13.1-x64.deb

读书方法论
------------
+ 储备型学习
    + 没有行为强化，没有目标的阅读，很容易导致自己降低目标
+ 战备型
    + 读书需要明确目标，为了实用而学习
    + 需要别人或者事物来衡量你的东西，而不是闭门造车，夜郎自大般，自我评价；使用别人参照体系，会弥补自己不足
+ 非凡（音译）原则
    + 用自己的理解去讲给别人听
    + 切记不能使概念解释概念
+ 康内尔笔记法
    + 记录 -> 简化 -> 摘要栏 -> 思考提出的自己的见解 -> 思考栏 
    + 将笔记分为几个区域，这样更好的记录东西


知识管理实战
------------
+ 记录阅读来源（后续可以想知道的时候，更好找到对应的人，来获得他的进展）
+ 记录阅读地址
+ 记录阅读时间结构
